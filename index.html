📊 医学统计学方法

P 值解读

定义：P值：原假设H₀为真时，观察到当前或更极端结果的概率
ASA声明：2016年P值使用规范
常见误区：

  • P<0.05 ≠ 临床意义显著
  • P>0.05 ≠ 无差异，可能样本量不足
  • 多次检验会导致假阳性升高
校正方法： Bonferroni、 FDR、 Benjamini-Hochberg

置信区间 (CI)

定义：置信区间：参数估计的区间范围
优势：同时展示效应大小与精度
解读：

  • 95%CI不含无效值=统计显著
  • 区间越窄，估计精度越高
常用：率差、率比、均数差、风险比

数据前提检验

正态性： Shapiro-Wilk、 KS检验

正态性检验用于判断数据是否服从正态分布，是参数检验的前提条件。Shapiro-Wilk检验适用于小样本（n<50），KS检验适用于大样本。

  1. 菜单：分析 → 描述统计 → 探索
  2. 将检验变量选入"因变量列表"
  3. 点击"绘制"，勾选"带检验的正态图"
  4. 点击确定，查看Shapiro-Wilk检验结果
  5. p>0.05说明数据服从正态分布
R
# Shapiro-Wilk正态性检验 shapiro.test(data$variable) # KS检验 ks.test(data$variable, "pnorm", mean=mean(data$variable), sd=sd(data$variable))
Python
from scipy import stats import pandas as pd # Shapiro-Wilk检验 stat, p = stats.shapiro(data['variable']) print(f'Shapiro-Wilk: 统计量={stat:.3f}, p值={p:.4f}') # KS检验 stat, p = stats.kstest(data['variable'], 'norm', args=(data['variable'].mean(), data['variable'].std())) print(f'KS检验: 统计量={stat:.3f}, p值={p:.4f}')

方差齐性： Levene、 Bartlett
数据类型：
  • 定量：连续、离散
  • 定性：二分类、多分类、有序
  • 生存数据：含时间与截尾
  • 重复测量：多时间点观测

回归模型

回归模型： 线性、 Logistic、 Cox

回归模型用于探索变量之间的关系，线性回归用于连续因变量，Logistic用于二分类因变量，Cox用于生存数据的风险因素分析。

线性回归

  1. 菜单：分析 → 回归 → 线性
  2. 将因变量、自变量分别选入对应框
  3. 方法选择"进入"（或逐步）
  4. 点击确定查看结果

Logistic回归

  1. 菜单：分析 → 回归 → 二元Logistic
  2. 将二分类因变量、协变量选入对应框
  3. 点击确定查看OR值及显著性

Cox回归

  1. 菜单：分析 → 生存分析 → Cox回归
  2. 时间变量、状态变量选入对应框
  3. 协变量选入协变量框
  4. 点击确定
R
# 线性回归 linear_model <- lm(y ~ x1 + x2, data = data) summary(linear_model) # Logistic回归 logit_model <- glm(group ~ x1 + x2, data = data, family = binomial()) summary(logit_model) exp(coef(logit_model)) # 查看OR值 # Cox回归 library(survival) cox_model <- coxph(Surv(time, status) ~ x1 + x2, data = data) summary(cox_model)
Python
import pandas as pd import numpy as np import statsmodels.api as sm from lifelines import CoxPHFitter # 线性回归 X = sm.add_constant(data[['x1', 'x2']]) y = data['y'] linear_model = sm.OLS(y, X).fit() print(linear_model.summary()) # Logistic回归 logit_model = sm.Logit(data['group'], X).fit() print(logit_model.summary()) print(np.exp(logit_model.params)) # OR值 # Cox回归 cph = CoxPHFitter() cph.fit(data, duration_col='time', event_col='status') cph.print_summary()

纵向数据： GEE、 线性混合模型
因果推断： PSM、 IPTW、工具变量、断点回归、双重差分
其他： 中介分析、 多重插补、竞争风险模型、结构方程模型

基础统计检验

定量数据： t检验、 ANOVA、非参数检验

t检验用于两组定量数据的比较，ANOVA用于多组比较，非参数检验用于不满足正态性的数据。

独立样本t检验

  1. 菜单：分析 → 比较均值 → 独立样本T检验
  2. 检验变量选入，分组变量选入并定义组
  3. 点击确定

单因素ANOVA

  1. 菜单：分析 → 比较均值 → 单因素ANOVA
  2. 因变量、因子选入对应框
  3. 事后检验选择LSD或Tukey
  4. 点击确定
R
# 独立样本t检验 t.test(y ~ group, data = data) # 单因素ANOVA aov_model <- aov(y ~ group, data = data) summary(aov_model) TukeyHSD(aov_model) # 事后检验 # 非参数：Wilcoxon秩和检验 wilcox.test(y ~ group, data = data)
Python
from scipy import stats import statsmodels.api as sm from statsmodels.formula.api import ols # 独立样本t检验 group1 = data[data['group']==0]['y'] group2 = data[data['group']==1]['y'] stat, p = stats.ttest_ind(group1, group2) print(f't检验: t={stat:.3f}, p={p:.4f}') # 单因素ANOVA model = ols('y ~ group', data=data).fit() anova_table = sm.stats.anova_lm(model, typ=2) print(anova_table) # 非参数检验 stat, p = stats.mannwhitneyu(group1, group2) print(f'Wilcoxon检验: stat={stat:.3f}, p={p:.4f}')

定性数据： 卡方检验、 Fisher确切概率、McNemar检验
相关分析： Pearson、 Spearman、Kendall相关
一致性： ICC、 Kappa

诊断试验评价

核心指标： 灵敏度、特异度、约登指数
预测值：PPV、NPV、似然比
综合评价： ROC曲线、AUC、校准曲线
报告规范： STARD

📈 WHO 传染病建模官方指南

📈 传染病数学建模

传染病建模是预测疾病传播、评估防控措施的核心方法，广泛应用于新冠、流感、登革热等传染病研究。

一、核心基础概念

S(t)：易感者 | E(t)：潜伏者 | I(t)：感染者 | R(t)：康复者 | M(t)：死亡者
β：传染率 | σ：潜伏转染率 | γ：康复率 | ξ：免疫丧失率 | μ：死亡率
dX/dt = 人群每日变化量 | R₀=β/γ：基本再生数

二、四大经典模型公式

1. SIR 模型

dS/dt=-βSI/N   dI/dt=βSI/N-γI   dR/dt=γI

2. SEIR 模型

dS/dt=-βSI/N   dE/dt=βSI/N-σE   dI/dt=σE-γI   dR/dt=γI

3. SEIRS 模型

dS/dt=-βSI/N+ξR   dE/dt=βSI/N-σE   dI/dt=σE-γI   dR/dt=γI-ξR

4. MSEIR 模型

dS/dt=-βSI/N   dE/dt=βSI/N-σE   dI/dt=σE-(γ+μ)I   dR/dt=γI   dM/dt=μI

三、参数速查表

符号名称计算方法权威参考
β传染率接触率×感染概率CDC参数标准
σ潜伏率1/潜伏期临床流行病学数据
γ康复率1/传染期疾病自然史
R₀再生数β/γWHO官方定义

四、Python 实战代码

依赖安装：pip install numpy scipy matplotlib

Python
# SEIR传染病模型完整实战代码 import numpy as np from scipy.integrate import odeint import matplotlib.pyplot as plt # 基础参数 N = 10000 # 总人口 E0, I0, R0 = 0, 1, 0 S0 = N - E0 - I0 - R0 beta, sigma, gamma = 0.3, 1/5, 1/10 t = np.linspace(0, 150, 150) # SEIR微分方程 def seir(y, t, N, beta, sigma, gamma): S, E, I, R = y dSdt = -beta * S * I / N dEdt = beta * S * I / N - sigma * E dIdt = sigma * E - gamma * I dRdt = gamma * I return dSdt, dEdt, dIdt, dRdt # 求解并绘图 result = odeint(seir, [S0, E0, I0, R0], t, args=(N, beta, sigma, gamma)) S, E, I, R = result.T plt.plot(t, S, label='易感者') plt.plot(t, E, label='潜伏者') plt.plot(t, I, label='感染者', c='red') plt.plot(t, R, label='康复者') plt.xlabel('天数'); plt.ylabel('人数') plt.title('SEIR传染病模型模拟曲线') plt.legend(); plt.show()

五、常见问题

✅ 模型不收敛：调整参数范围/优化求解器
✅ 结果不符合实际：重新校准参数
✅ 无真实数据：使用文献报道的R₀值

📝 临床研究设计

证据等级：RCT > 队列 > 病例对照 > 横断面

观察性研究：
• 横断面：患病率调查
• 病例对照：回顾性，计算OR
• 队列：前瞻性，计算RR，遵循STROBE

实验性研究：
• RCT：遵循CONSORT
• 诊断试验：遵循STARD
量表研究：遵循COSMIN

🏥 真实世界研究 (RWS)

定义：真实世界研究：基于常规诊疗数据的临床研究
数据来源：EHR、医保、登记库、区域健康数据

混杂控制：PSM、IPTW、DID、工具变量、高维倾向得分
报告规范：RECORD
优势：外推性强、样本量大、覆盖广泛人群

📊 量表信效度分析

信度：Cronbach's α、重测信度、分半信度
效度：内容效度、结构效度、效标效度
结构分析：EFA、CFA

R
library(psych) library(lavaan) # Cronbach's α alpha(data) # 探索性因子分析(EFA) fa_result <- fa(data, nfactors=3, rotate="varimax") print(fa_result) # 验证性因子分析(CFA) model <- ' F1 =~ item1 + item2 + item3 F2 =~ item4 + item5 + item6 ' fit <- cfa(model, data=data) summary(fit, fit.measures=TRUE)

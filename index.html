📊 医学统计学方法

P 值解读

定义：P值：原假设H₀为真时，观察到当前或更极端结果的概率
常见误区：

  • P<0.05 ≠ 临床意义显著
  • P>0.05 ≠ 无差异，可能样本量不足
  • 多次检验导致假阳性升高
校正方法：Bonferroni校正、FDR、Benjamini-Hochberg法

置信区间 (CI)

定义：参数估计的区间范围
优势：同时展示效应大小与精度

  • 95%CI不含无效值 = 统计显著
  • 区间越窄，估计精度越高
常用：率差、率比、均数差、风险比

数据前提检验

正态性：Shapiro-Wilk检验、KS检验

正态性检验判断数据是否服从正态分布，是参数检验的前提。Shapiro-Wilk适用于小样本(n<50)，KS检验适用于大样本。

  1. 分析 → 描述统计 → 探索
  2. 变量选入"因变量列表"
  3. 点击"绘制"，勾选"带检验的正态图"
  4. 确定后查看Shapiro-Wilk结果
  5. p>0.05 服从正态分布
R
shapiro.test(data$variable) ks.test(data$variable, "pnorm", mean=mean(data$variable), sd=sd(data$variable))
Python
from scipy import stats stat, p = stats.shapiro(data['variable']) print(f'Shapiro-Wilk: 统计量={stat:.3f}, p值={p:.4f}') stat, p = stats.kstest(data['variable'], 'norm', args=(data['variable'].mean(), data['variable'].std())) print(f'KS检验: 统计量={stat:.3f}, p值={p:.4f}')

方差齐性：Levene检验、Bartlett检验
数据类型：
  • 定量：连续、离散
  • 定性：二分类、多分类、有序
  • 生存数据：含时间与截尾
  • 重复测量：多时间点观测

回归模型

类型：线性回归、Logistic回归、Cox回归

线性回归用于连续因变量，Logistic用于二分类因变量，Cox用于生存数据的风险因素分析。

线性回归

  1. 分析 → 回归 → 线性
  2. 因变量、自变量选入对应框
  3. 方法选"进入"
  4. 确定

Logistic回归

  1. 分析 → 回归 → 二元Logistic
  2. 二分类因变量、协变量选入
  3. 确定查看OR值

Cox回归

  1. 分析 → 生存分析 → Cox回归
  2. 时间、状态变量、协变量选入
  3. 确定
R
linear_model <- lm(y ~ x1 + x2, data = data) summary(linear_model) logit_model <- glm(group ~ x1 + x2, data = data, family = binomial()) summary(logit_model) exp(coef(logit_model)) library(survival) cox_model <- coxph(Surv(time, status) ~ x1 + x2, data = data) summary(cox_model)
Python
import numpy as np import statsmodels.api as sm from lifelines import CoxPHFitter X = sm.add_constant(data[['x1', 'x2']]) y = data['y'] model = sm.OLS(y, X).fit() print(model.summary()) logit = sm.Logit(data['group'], X).fit() print(logit.summary()) cph = CoxPHFitter() cph.fit(data, duration_col='time', event_col='status') cph.print_summary()

纵向数据：GEE、线性混合模型
因果推断：PSM、IPTW、工具变量、断点回归、双重差分
其他：中介分析、多重插补、竞争风险模型、结构方程模型

定量数据比较

方法：t检验、ANOVA、非参数检验

t检验用于两组比较，ANOVA用于多组比较，非参数检验用于不满足正态性的数据。配对设计用配对t检验。

独立样本t检验

  1. 分析 → 比较均值 → 独立样本T检验
  2. 检验变量选入，分组变量定义组
  3. 确定

单因素ANOVA

  1. 分析 → 比较均值 → 单因素ANOVA
  2. 因变量、因子选入
  3. 事后检验选LSD或Tukey
  4. 确定
R
t.test(y ~ group, data = data) t.test(before, after, paired = TRUE) aov_model <- aov(y ~ group, data = data) summary(aov_model) TukeyHSD(aov_model) wilcox.test(y ~ group, data = data) kruskal.test(y ~ group, data = data)
Python
from scipy import stats import statsmodels.api as sm from statsmodels.formula.api import ols g1 = data[data['group']==0]['y'] g2 = data[data['group']==1]['y'] stat, p = stats.ttest_ind(g1, g2) print(f't检验: t={stat:.3f}, p={p:.4f}') model = ols('y ~ group', data=data).fit() print(sm.stats.anova_lm(model, typ=2)) stat, p = stats.mannwhitneyu(g1, g2) print(f'Wilcoxon: stat={stat:.3f}, p={p:.4f}')

定性数据检验

方法：卡方检验、Fisher确切概率法、McNemar检验

卡方检验用于分类数据关联性分析。2×2表用Pearson卡方；期望频数<5时用Fisher确切概率法；配对设计用McNemar检验。

  1. 分析 → 描述统计 → 交叉表
  2. 行变量、列变量选入
  3. 点击"统计"，勾选"卡方"
  4. 期望频数<5时勾选"Fisher精确检验"
  5. 确定
R
chisq.test(table) fisher.test(table) mcnemar.test(table)
Python
from scipy.stats import chi2_contingency, fisher_exact, mcnemar chi2, p, dof, exp = chi2_contingency(table) print(f'卡方={chi2:.3f}, p={p:.4f}') oddsratio, p = fisher_exact(table) print(f'OR={oddsratio:.3f}, p={p:.4f}') result = mcnemar(table, exact=True) print(f'McNemar: p={result.pvalue:.4f}')

相关与一致性分析

相关分析：Pearson、Spearman、Kendall

Pearson用于正态连续变量，Spearman用于非正态或有序数据，Kendall对小样本更稳健。

R
cor.test(data$x, data$y, method = "pearson") cor.test(data$x, data$y, method = "spearman") cor.test(data$x, data$y, method = "kendall")
Python
from scipy.stats import pearsonr, spearmanr, kendalltau r, p = pearsonr(data['x'], data['y']) r, p = spearmanr(data['x'], data['y']) r, p = kendalltau(data['x'], data['y'])

一致性：ICC（连续变量）、Kappa（分类变量）

ICC评价连续变量测量一致性（不同评分者/仪器），Kappa评价分类变量一致性（诊断结果）。

📈 传染病数学建模

传染病建模是预测疾病传播、评估防控措施的核心方法。

一、核心概念

S(t)：易感者 | E(t)：潜伏者 | I(t)：感染者 | R(t)：康复者 | M(t)：死亡者
β：传染率 | σ：潜伏转染率 | γ：康复率 | ξ：免疫丧失率 | μ：死亡率
dX/dt = 人群每日变化量 | R₀=β/γ：基本再生数

二、四大经典模型

1. SIR

dS/dt=-βSI/N   dI/dt=βSI/N-γI   dR/dt=γI

2. SEIR

dS/dt=-βSI/N   dE/dt=βSI/N-σE   dI/dt=σE-γI   dR/dt=γI

3. SEIRS

dS/dt=-βSI/N+ξR   dE/dt=βSI/N-σE   dI/dt=σE-γI   dR/dt=γI-ξR

4. MSEIR

dS/dt=-βSI/N   dE/dt=βSI/N-σE   dI/dt=σE-(γ+μ)I   dR/dt=γI   dM/dt=μI

三、参数表

符号名称计算方法参考
β传染率接触率×感染概率CDC
σ潜伏率1/潜伏期临床数据
γ康复率1/传染期疾病自然史
R₀再生数β/γWHO

四、Python 实战代码

依赖：pip install numpy scipy matplotlib

Python
import numpy as np from scipy.integrate import odeint import matplotlib.pyplot as plt N = 10000 E0, I0, R0 = 0, 1, 0 S0 = N - E0 - I0 - R0 beta, sigma, gamma = 0.3, 1/5, 1/10 t = np.linspace(0, 150, 150) def seir(y, t, N, beta, sigma, gamma): S, E, I, R = y dSdt = -beta * S * I / N dEdt = beta * S * I / N - sigma * E dIdt = sigma * E - gamma * I dRdt = gamma * I return dSdt, dEdt, dIdt, dRdt result = odeint(seir, [S0, E0, I0, R0], t, args=(N, beta, sigma, gamma)) S, E, I, R = result.T plt.plot(t, S, label='易感者') plt.plot(t, E, label='潜伏者') plt.plot(t, I, label='感染者', c='red') plt.plot(t, R, label='康复者') plt.xlabel('天数') plt.ylabel('人数') plt.title('SEIR传染病模型模拟') plt.legend() plt.show()

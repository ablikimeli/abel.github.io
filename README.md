# 医学统计与生物信息学习平台 | Abel医研统计

## 📖 平台介绍
这是一个**一站式医学科研学习平台**，整合了医学统计、生物信息学、临床研究、量表评估、公共数据库、科研工具等全流程的科研资源，为医学科研工作者提供全面的学习和工具支持。

平台所有外部链接均为国内可正常访问的权威资源，移除所有无法访问的维基百科链接，确保您可以正常获取所有资源。

## ✨ 核心功能
### 1. 完整的统计方法教程
- 从基础的t检验、ANOVA到高级的回归、因果推断、贝叶斯统计、Meta分析，覆盖所有常用统计方法
- 每个方法都提供：
  - 概念简介
  - SPSS操作流程（文字步骤）
  - 带一键复制功能的R代码
  - 带一键复制功能的Python代码
  - 案例解读与结果报告模板
- 额外包含：缺失数据处理（MICE多重插补）、时间序列分析（ARIMA）、竞争风险模型、中介分析、聚类分析、PCA、一致性分析等
- 所有统计方法链接均为权威教程，国内可正常访问

### 2. 统计公式速查表
- 涵盖单样本t、独立t、配对t、卡方、Pearson r、线性回归F、Logistic OR、灵敏度、特异度等核心公式
- 附带使用条件说明，方便快速查阅

### 3. 研究报告规范检查清单
- STROBE（观察性研究）、CONSORT（RCT）、PRISMA（系统评价/Meta分析）
- STARD（诊断试验）、TRIPOD（预测模型）、SPIRIT（试验方案）
- RECORD（真实世界数据）、COSMIN（量表）、ARRIVE（动物实验）
- EQUATOR报告规范总库入口

### 4. 主要医学期刊统计要求
- NEJM、JAMA、Lancet、BMJ、Nature Medicine 的统计报告规范
- 中文核心期刊统计要求
- 通用投稿统计建议

### 5. 常见统计错误与避坑指南
- 8个最常见统计错误：P值滥用、多重比较、前提检验、亚组分析、混杂控制、缺失数据、样本量、选择性报告
- 每项含错误示例、正确做法和具体建议

### 6. 生物信息学分析

- 附带完整的可复制的R/Python分析代码，直接用于您的研究

### 7. 常用医学量表汇总

### 8. 传染病数学建模

### 9. 临床研究资源

## 📱 使用说明
1. **快速跳转**：点击顶部导航栏或左侧TOC目录，快速跳转到对应模块
2. **模块展开**：点击大模块标题（如"医学公共数据库"），展开查看详细内容
3. **方法教程**：点击统计方法的"查看详细操作"按钮，展开查看完整教程和代码（支持方法介绍/SPSS/R/Python/案例解读标签页切换）
4. **代码复制**：所有代码支持一键复制，点击代码右上角"复制"按钮即可
5. **深色模式**：点击右上角月亮/太阳图标切换深色/浅色主题
6. **搜索功能**：Ctrl+K 或点击搜索图标，可搜索全站内容
7. **外部链接**：所有外部链接均为国内可正常访问的权威资源
